# Sistema Financeiro para Pizzaria

Sistema em desenvolvimento em Java para gerenciamento financeiro de uma pizzaria.

## Funcionalidades (em desenvolvimento)

- Login de administrador e funcionário
- Controle de usuários
- Cadastro de produtos
- Registro de compras
- Registro de vendas
- Controle de contas
- Relatórios financeiros

## Tecnologias

- Java 17
- Maven
- IntelliJ IDEA
- MySQL (futuramente)
- Git e GitHub
